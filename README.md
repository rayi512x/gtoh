# gtoh
Converts the letter G to H

# Building
    c++ gtoh.cpp -o gtoh

# Usage
Read from arguments:

	./gtoh "Enter a sentence that contains the letter G"
or

    ./gtoh Enter a sentence that contains the letter G

Output: ```Enter a sentence that contains the letter H```
